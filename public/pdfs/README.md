# PDF Files Directory

Place your book PDF files in this directory with the following naming convention:

- `quantum-computing-explained.pdf` - Quantum Computing Explained
- `the-martian.pdf` - The Martian
- `to-kill-a-mockingbird.pdf` - To Kill a Mockingbird
- `pride-and-prejudice.pdf` - Pride and Prejudice
- `the-catcher-in-the-rye.pdf` - The Catcher in the Rye
- `the-great-gatsby.pdf` - The Great Gatsby
- `1984.pdf` - 1984

The PDFs will be accessible at: `http://localhost:4000/pdfs/[filename]`

After adding PDFs, make sure to:
1. Run `npm run seed:books` to update the database with PDF URLs
2. Restart your server if it's running
